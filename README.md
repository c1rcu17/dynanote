# Dynanote

Take powerful notes with markdown and handlebars and save them to your Google Drive

### Website:

https://c1rcu17.github.io/dynanote/

### Build

Create a config file in `src/config/config.json`

```json
{
  "google": {
    "clientId": "",
    "apiKey": ""
  }
}
```

```sh
npm run build
```

### Serve

```sh
npm start
```
